# Trello Card Template

## Purpose

Use this template to prepare Trello card drafts from AI QA outputs.

Important rule:

This is a draft template only.

AI agents must not create Trello cards without explicit human approval.

---

## 1. Card Metadata

| Field | Value |
|---|---|
| Board | |
| List | QA To Do / In Review / Blocked / Done |
| Priority | P0 / P1 / P2 / P3 |
| Label | QA / Bug / Risk / API / DB / Web / Mobile |
| Owner | |
| Source Agent | |
| Human Approval Required | Yes |

---

## 2. Card Title

Write a short and clear title.

Example:

Login QA - Generate edge and negative test cases

---

## 3. Description

Describe what needs to be done.

Include:

- Feature
- Scope
- Platform
- Expected output
- Risk if not completed

---

## 4. Checklist

- [ ] Requirement reviewed
- [ ] Test plan prepared
- [ ] Happy path cases prepared
- [ ] Edge cases prepared
- [ ] Negative cases prepared
- [ ] API validation prepared
- [ ] DB validation prepared
- [ ] Output reviewed
- [ ] Human approval completed
